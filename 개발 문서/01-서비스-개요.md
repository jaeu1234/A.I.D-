# 01 · 서비스 개요

## 한 줄 정의
**선생님 위치 안내 시스템** — 학생이 "지금 이 선생님이 학교 건물 어디에 계신지"를
2D 평면도 위에서 시각적으로 확인하는 학교 내부용 웹서비스.

## 무엇을 / 누구를 위해
- **대상**: 학생(공개 열람), 선생님·관리자(일정 관리)
- **핵심 가치**: "지금 이 선생님 어디 계세요?"라는 질문에 평면도 + 핀으로 즉답
- **주 사용처**: 복도 전자칠판, 학생 개인 기기

## 주요 화면
| 화면 | 파일 | 기능 |
|------|------|------|
| 학생 열람 | `index.html` | 선생님 검색 / 층별 평면도 / 현재+다음 위치 / 요일 동선 타임라인 |
| 관리자 | `admin.html` | PIN 로그인 / 날짜별 임시 일정 등록·삭제 / 시간표 요약 |
| 업로드 | `upload.html` | 시간표 사진 → Claude 비전 API 분석 → 결과 수정 → 저장 |

## 기술 스택
- **언어**: 바닐라 HTML/CSS/JavaScript (빌드 도구 없음, ES module import)
- **렌더링**: Canvas 2D API (평면도·핀·동선)
- **AI**: Anthropic Claude API (시간표 사진 분석) — 모델 `claude-sonnet-5`
- **저장**: `localStorage` (임시일정·AI시간표·API키)
- **호스팅**: Vercel — 프로덕션 **https://teacher-map.vercel.app/** (빌드 없이 정적 파일 그대로 서빙).
  단 ES module 때문에 `file://` 직접 열기는 불가, 반드시 HTTP 서버로 서빙해야 함
  (로컬 확인은 `서버-실행.bat` 또는 임의의 정적 서버).

## 데이터 우선순위 (위치 계산 시)
```
임시 일정  (localStorage: teacher_overrides)
    ↓ 없으면
AI 분석 시간표 (localStorage: teacher_schedules)
    ↓ 없으면
기본 하드코딩 시간표 (src/data/schedule.js)
```
`src/lib/location.js`의 `getTeacherLocation()`이 이 순서를 구현한다.
